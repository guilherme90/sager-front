### Sager - Front ###

Simple project to stater React project with Webpack hot middleware, react-hot-loader and more!

#### Packages included ####
- React
- Bootstrap 3
- ES6
- Webpack 2
- Axios

#### Dev Server ####
```sh
$ npm run dev-server
```

#### Prod Server ####
```sh
$ npm run build
$ npm run prod-server
```
